GITHUB

Github é uma plataforma de controle de versão de código onde podemos compartilhar e trabalhar projetos de forma mais produtiva e dinâmica. O gerenciamento dos códigos é feito através dos repositórios que podem ser públicos ou privados. para iniciar antes de tudo você precisa ter uma conta no Github. Acesse o link abaixo e faça a sua.

Acesse Github aqui!

Criação da Chave de Segurança

Para utilizar o Github deve-se criar uma chave de segurança. As chaves SSH permitem comunicação segura entre você e Github. Normalmente as chaves ficam no subdiretório do usuário da máquina denominado .ssh. Caso você já tenha chaves e queira gerar outras faça o backup dos arquivos desta pasta, prioritariamente id_rsa e id_rsa.pub. Depois delete estes dois arquivos para prosseguir.

Para gerar as chaves você deve seguir os seguintes passos (comandos digitados no terminal ou prompt):

[1] - Gerar a chave (substitua email@email.com.br pelo seu e-mail):

ssh-keygen -t rsa -C "email@email.com.br"

[2] - Preencha uma passphrase (opcional):

Enter passphrase (empty for no passphrase): [Type a passphrase]
Enter same passphrase again: [Type passphrase again]

[3] - Você verá algo como:

Your identification has been saved in /your_home_path/.ssh/id_rsa.
Your public key has been saved in /your_home_path/.ssh/id_rsa.pub.
The key fingerprint is:
01:0f:f4:3b:ca:85:d6:17:a1:7d:f0:68:9d:f0:a2:db email@email.com.br

[4] - Agora você terá que adicionar a sua chave no site github.
  • Faça Login no Github.
  • Acesse Account Settings - Canto superior direito.
  • Clique em SSH Keys.
  • Cole sua chave no campo KEY (a chave está no arquivo id_rsa.pub).
  • Confirme a ação digitando sua senha do Github.

PRONTO! SUA CHAVE ESTÁ ADICIONADA NO GITHUB!!!

Agora você pode testar sua chave com o seguinte comando:

ssh -T git@github.com

Você verá uma mensagem parecida com esta:

Hi username! You've successfully authenticated, but GitHub does not provide shell access.

Seu primeiro commit no Github

Antes de mandar um código pro Github você precisa criar um repositório. Para isto faça login no Github e acesse Create a Repository. Crie um novo repositório para o projeto.

Feito isto acesse o terminal, acesse o diretório do projeto que deseja enviar e digite os 5 comandos abaixo:

[1] git init
[2] git add .
[3] git commit -m "Meu primeiro commit"
[4] git remote add origin git@github.com:USUARIO-GITHUB/REPOSITORIO.git
[5] git push origin master

Explicando os comandos: O primeiro apenas inicia o GIT no diretório do projeto. O segundo comando adiciona todos os arquvos no diretório e subdiretórios do seu projeto. O terceiro dá um título para o commit. O quarto comando faz a ligação com o GIT (substitua USUARIO pelo seu nome de usuario no GITHUB e REPOSITORIO pelo nome do repositório criado). Este comando só precisa ser digitado na primeira vez que você for subir o projeto. Nas modificações subsequentes parte para o comando 5 direto. Finalmente o último comando faz o UPLOAD do seu projeto para o Github. Será solicitado seu nome de usuário e senha do Github para concluir a operação.
